Farsi Gradient Text

Making gradient text sometimes is hectic. While it is supported in some new browsers there are some other older browsers that cannot create gradient text correctly.
Thus there are some online services like http://patorjk.com/text-color-fader/ that receive a sample text and create a series of SPAN tags with different text colors.
This kind of services works fine for latin text that their characters are separated naturally. For example, The header of this page was created with this technique.
In the following, there are 2 sentences. The first one was created with some CSS technique that at the moment of writing this text just supported in Google Chrome and Edge browsers.
Therefore if you see this text as a simple white text the reason is that is not supported by your browser.
The second try was made by using online services and it shows how this services cannot create a correct right to left sentence with their technique.
The third try was made with the library that I created and it generates a correct gradient sentence for a right to left sentence.
You can find a sample of correct usage of this library in the example inside of this page.
